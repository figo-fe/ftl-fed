欢迎使用ftl-fed V1.0

1. 使用说明

使用前先在mock/config.json 中配置ftl路径，静态文件放在static目录

======================================

mock目录为本地调试数据

config.json配置ftl路径以及ftl全局对象

        {
            "ftlRoot": "./demo/",
            "globalData":{
                "global": "powered by Figo(hanlongfei@sogou-inc.com)"
            }
        }

urlmap.json 配置 url路由，以及ajax代理

        {
            "ajax":[
                "/ajax/test.html"
            ],
            "route":{
                "/index": "/index.html"
            }
        }

所有在ajax数组中的url都会按照下面规则代理到ajax目录中json文件

/ajax/test.html => data_ajax_test_html.json ([url中 ".?/=-&" 替换为 '_')
